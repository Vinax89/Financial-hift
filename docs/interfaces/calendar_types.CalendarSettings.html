    Interface CalendarSettings

    Calendar display settings configuration

    Remarks

    Controls which types of events are visible on the calendar and whether weekends are displayed. Users can toggle these settings to customize their calendar view.

    Example

    const settings: CalendarSettings = {
      showWeekends: true,
      showIncome: true,
      showExpenses: true,
      showBills: true,
      showGoals: false,
      showShifts: false
    };
